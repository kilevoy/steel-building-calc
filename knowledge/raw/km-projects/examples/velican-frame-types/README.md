# Типовые виды каркасов VELICAN

Эта папка предназначена для локального хранения PDF-схем с типовыми вариантами каркасов VELICAN.

Файлы используются как практический источник для будущего раздела "Единое здание":

- понимание типов кровли;
- сравнение односкатных и двускатных схем;
- проверка многопролётных схем;
- анализ вариантов с антресолью;
- подготовка обезличенных входных сценариев для wiki и acceptance tests.

PDF-файлы не коммитятся в публичный GitHub. В Git хранится только этот README и записи в `knowledge/raw/source-register.md`.

## Локальные файлы

- `Великан односкатный.pdf`
- `Великан односкатный 2 пролета.pdf`
- `Великан 2 пролета.pdf`
- `Великан 3 пролета.pdf`
- `Великан 3 пролета вдоль.pdf`
- `Великан 4 пролета.pdf`
- `Великан с антресолью.pdf`

## Правила работы

- Не считать PDF автоматическим oracle для расчётных формул.
- Использовать как источник схем, компоновок и практических сценариев.
- Извлечённые параметры переносить в `knowledge/wiki/` в обезличенном виде.
- Если появляется числовой acceptance-сценарий, указывать ID источника из реестра.
