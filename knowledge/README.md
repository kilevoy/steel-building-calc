# Инженерная память проекта

`knowledge/` хранит инженерный контекст проекта `steel-building-calc`: структуру модулей, принятые решения, временные допущения, связь с Excel/VELICAN-оракулами и правила работы для будущих AI-агентов.

Эта папка не заменяет исходный код и тесты. Код остаётся источником реализации, тесты — автоматической проверкой поведения, а `knowledge/` — источником понимания: почему решения были приняты, что уже проверено и что требует отдельной инженерной верификации.

`knowledge/raw/` предназначена для описания исходных материалов: Excel, КМ-проектов, нормативных документов и скриншотов. Тяжёлые и конфиденциальные файлы по умолчанию не коммитятся.
