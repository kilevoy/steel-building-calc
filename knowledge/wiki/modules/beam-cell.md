# Beam Cell

Модуль балки покрытия используется для расчёта соответствующего конструктивного элемента здания.

Текущий статус: требуется расширение smoke/acceptance tests. При изменениях проверять входные параметры, единицы измерения, связь с геометрией здания и отсутствие изменений в расчётных формулах без отдельного задания.
