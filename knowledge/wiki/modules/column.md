# Column

Модуль колонны рассчитывает подбор/проверку колонн по входным параметрам здания и нагрузкам.

Известно: default input вынесен в `src/defaults/columnInput.ts`, а smoke-тест запускает `runCalculation(DEFAULT_COLUMN_INPUT)` и проверяет непустой результат.

Модуль связан с нагрузками от здания и с допущением `H = 0`. При изменениях нужно сверять результаты, тесты и предупреждения в UI. Формулы не менять без отдельного инженерного решения.
