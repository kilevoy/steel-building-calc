# Открытые допущения

## Горизонтальная реакция H

Горизонтальная реакция `H` пока временно принимается равной `0`. Это допущение влияет на расчёт колонн.

Нельзя считать его окончательным инженерным решением. Требуется отдельная проверка методики расчёта `H`. UI должен показывать предупреждение.

## Axial wind load в Purlins

В purlins есть временное предупреждение по axial wind load. Это нужно сохранить как открытый вопрос.

Нельзя удалять предупреждение без инженерной проверки.
