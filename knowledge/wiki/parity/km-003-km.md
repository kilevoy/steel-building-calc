# KM-003 — обезличенный КМ-проект 2

Источник: `KM-003` из `knowledge/raw/source-register.md`.

Файл: `KM.pdf`.

Статус ingest: выполнено частичное текстовое извлечение из PDF. Из-за особенностей PDF-шрифтов часть страниц извлекается как `(cid:...)`, поэтому сведения требуют визуальной проверки по PDF.

## Назначение

КМ-проект используется как практический инженерный пример. Он не является oracle для расчётных формул, но полезен для проверки типовых схем каркаса, ферм, балок покрытия, распорок, фахверка, фундаментов и монтажных узлов.

## Обезличенные признаки проекта

- Тип объекта по извлечённым подписям: гараж.
- Габариты по извлечённой подписи: примерно `24 x 42 x 7`.
- Количество страниц PDF: 36.
- Встречаются схемы баз колонн, элементов каркаса, ферм, балок покрытия, распорок и связей.

## Наблюдаемый состав листов

По извлечённым подписям видны:

- общие указания;
- общий вид;
- схема расположения баз колонн и фундамента под наружные стены;
- узлы и расчётные нагрузки на фундаменты;
- схема расположения элементов каркаса на отметке 0,000;
- схема расположения ферм, балок покрытия, распорок и горизонтальных связей по верхним поясам ферм;
- схема расположения распорок по нижним поясам ферм и вертикальных связей между фермами.

## Наблюдаемые инженерные элементы

- колонны и фермы;
- балки покрытия;
- горизонтальные связи по покрытию;
- вертикальные связи по фермам и колоннам;
- распорки;
- колонны фахверка;
- рамы ворот;
- базы колонн;
- анкерные болты;
- ленточный фундамент под наружные стены.

## Нормативные и производственные указания

В общих указаниях видны ссылки на СП 20.13330.2016, СП 131.13330.2020, СП 70.13330.2012, ГОСТ 23118-2019, ГОСТ 27772-2021, СП 28.13330.2017, СП 16.13330.2017 и монтажные требования.

Эти ссылки зафиксированы как наблюдаемые подписи КМ-проекта. Нормативные формулы и коэффициенты должны проверяться по исходным нормативным документам отдельно.

## Связь с калькулятором

Источник полезен для будущей проверки:

- количества рам и ферм;
- расположения фахверковых колонн;
- схемы связевых блоков;
