# SCN-KM-001 — единое здание по КМ-примеру

Статус: `draft`.

Источники: `KM-002`, `KM-003`.

Назначение: подготовить практический сценарий для будущего раздела единого здания.

## Входные данные

Требуется выбрать один обезличенный КМ-пример и извлечь:

- тип здания;
- пролёт;
- длину;
- высоту;
- шаг рам;
- количество осей;
- тип кровли;
- наличие фахверка;
- наличие связевых блоков;
- состав элементов: колонны, фермы, балки покрытия, прогоны, связи.

## Ожидаемые выходы

Для будущего Summary требуется проверить:

- количество рам;
- количество ферм;
- количество крайних и средних колонн;
- количество фахверковых колонн;
- наличие балок покрытия;
- наличие связей и распорок.

## Открытые вопросы

- Визуально проверить страницы PDF, где текст извлекается как `(cid:...)`.
- Не переносить конфиденциальные данные в Git.
