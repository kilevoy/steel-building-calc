# SCN-BEAM-CELL-001 — балочная клетка

Статус: `draft`.

Источник: `EXCEL-007`.

Назначение: подготовить сверку ВБ/ГБ/стоек по профилям, массе и стоимости.

## Входные данные

Требуется извлечь:

- габариты балочной клетки;
- размер вдоль ГБ;
- размер поперёк ГБ;
- высота К;
- пролёт ГБ;
- шаг ГБ;
- конструкция перекрытия;
- нагрузка на перекрытие;
- вид конструкции;
- максимальный шаг ВБ.

## Ожидаемые выходы

Требуется извлечь:

- профиль ВБ;
- профиль ГБ;
- профиль К, если применимо;
- общий вес ВБ;
- общий вес ГБ;
- итоговая масса;
- стоимость.

## Открытые вопросы

- Сопоставить текущий Beam Cell UI с терминами `ВБ`, `ГБ`, `К`.
