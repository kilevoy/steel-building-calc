# SCN-COLUMN-001 — колонна, первый Excel-oracle сценарий

Статус: `draft`.

Источник: `EXCEL-010`.

Назначение: подготовить первый минимальный acceptance-сценарий для модуля колонн.

## Входные данные

Требуется извлечь из `EXCEL-010` обезличенный набор входов:

- город;
- тип кровли;
- пролёт;
- длина;
- высота;
- уклон кровли;
- шаг рам;
- шаг стоек фахверка;
- количество пролётов;
- тип колонны;
- снеговая нагрузка;
- ветровая нагрузка;
- `N`;
- `M`.

## Ожидаемые выходы

Требуется извлечь:

- выбранный профиль;
- сталь;
- масса;
- максимальный коэффициент использования;
- критерий, который стал определяющим.

## Допуск

Требует определения после ручного чтения Excel и сопоставления с текущим engine.

## Не проверяется

Этот сценарий не должен проверять все профили и все формулы workbook.

## Открытые вопросы

- Сопоставить `Сводка` Excel с `DEFAULT_COLUMN_INPUT`.
- Проверить влияние `H = 0`.
- Проверить типы колонн: крайняя, средняя, фахверковая.
