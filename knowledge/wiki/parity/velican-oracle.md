# VELICAN oracle

Excel/VELICAN используется как oracle для сверки там, где это согласовано и зарегистрировано в `source-register.md`.

Oracle не должен изменяться под код. Если код не совпадает с oracle, расхождение нужно зафиксировать. Не каждое отличие автоматически является ошибкой: иногда причина в нормализации исходных данных, единицах измерения или разных входных допущениях.
