# KM-002 — обезличенный КМ-проект 1

Источник: `KM-002` из `knowledge/raw/source-register.md`.

Файл: `КМ1.pdf`.

Статус ingest: выполнено частичное текстовое извлечение из PDF. Из-за особенностей PDF-шрифтов часть страниц извлекается как `(cid:...)`, поэтому сведения требуют визуальной проверки по PDF.

## Назначение

КМ-проект используется как практический инженерный пример. Он не является oracle для расчётных формул, но полезен для проверки состава конструкций, типовых схем, маркировки элементов, связей, фахверка и фундаментов.

## Обезличенные признаки проекта

- Тип объекта по извлечённым подписям: утеплённый неотапливаемый склад.
- Габариты по извлечённой подписи: примерно `11,6 x 50,4 x 4,0(h)`.
- Количество страниц PDF: 39.
- Встречаются схемы каркаса, ферм, балок покрытия, распорок, горизонтальных и вертикальных связей.

## Наблюдаемый состав листов

По извлечённым подписям видны:

- общие указания;
- общий вид;
- узлы баз колонн;
- расчётные нагрузки на фундаменты;
- схема расположения элементов каркаса на отметке 0,000;
- схема расположения распорок по нижним поясам ферм и вертикальных связей по фермам;
- схема расположения стропильных ферм, балок покрытия, распорок и горизонтальных связей.

## Наблюдаемые инженерные элементы

- несущие рамы каркаса: колонны и фермы;
- горизонтальные и вертикальные связи;
- распорки;
- стойки фахверка;
- рамы ворот;
- кровельные прогоны из холодногнутых оцинкованных профилей;
- базы колонн с анкерными болтами;
- упоры и переходные пластины в связевых блоках;
- ленточный фундамент под наружные стены.

## Нормативные и производственные указания

В общих указаниях видны ссылки на СП 20.13330.2016, СП 131.13330.2020, СП 70.13330.2012, ГОСТ 23118-2019, ГОСТ 27772-2021, СП 28.13330.2017 и связанные требования к изготовлению, монтажу и антикоррозионной защите.

Эти ссылки зафиксированы как наблюдаемые подписи КМ-проекта. Их нельзя использовать как замену нормативной проверки методики расчёта.

## Связь с калькулятором

Источник полезен для будущего раздела единого здания:

- проверка состава элементов каркаса;
- проверка количества колонн, ферм, балок покрытия и связей;
- проверка того, как фахверк и связевые блоки отражаются в реальном КМ;
- проверка практических схем расположения элементов.

## Что требует проверки

- Визуально проверить страницы, где текст извлекается как `(cid:...)`.
- Сопоставить реальные марки элементов с текущими понятиями модулей: Column, Truss, Purlins, Beam Cell, Summary.
- Извлечь обезличенный сценарий геометрии здания для будущего smoke/acceptance test.
- Не переносить адреса, имена и служебные данные проекта в публичные wiki-страницы.
