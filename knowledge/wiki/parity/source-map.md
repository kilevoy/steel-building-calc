# Карта соответствия источников

Эта страница связывает расчётные модули приложения с источниками, которые нужно читать перед изменениями и использовать для acceptance/parity-сценариев.

## Таблица соответствия

| Модуль | Основные источники | Дополнительные источники | Что проверять |
|---|---|---|---|
| Column | `EXCEL-010` | `KM-002`, `KM-003`, `GH-001` | входы здания, тип колонны, `N/M`, профиль, масса, устойчивость, гибкость |
| Truss | `EXCEL-008` | `KM-002`, `KM-003`, `GH-001` | ВП, НП, решётка, масса фермы, реакции, открытый вопрос `H` |
| Purlins | `EXCEL-009` | `KM-002`, `KM-003` | прокатные/горячекатаные прогоны, ЛСТК МП350/МП390, шаг, масса, axial wind load |
| Beam Cell | `EXCEL-007` | `KM-002`, `KM-003` | ВБ, ГБ, стойки `К`, масса, стоимость, терминологическое соответствие UI |
| Window Riegel | `EXCEL-005` | `KM-002`, `KM-003` | тип окна, профиль ригеля, гибкость, прочность, вертикальный и горизонтальный прогиб |
| Crane Beam | `EXCEL-006` | `EXCEL-010` | крановые входы, рельс, усилия, прочность, устойчивость, усталость, прогибы |
| Climate | `GH-002` | `EXCEL-005`, `EXCEL-008`, `EXCEL-009`, `EXCEL-010` | город, `w0`, `Sg`, lazy loading климатических данных |
| Summary / Unified Building | `KM-002`, `KM-003`, `GH-001` | все Excel-oracle sources | состав здания, количество элементов, практичность сводки |

## Правила использования

Перед изменением модуля читать:

1. страницу модуля в `knowledge/wiki/modules/`;
2. соответствующую страницу источника в `knowledge/wiki/parity/`;
3. `knowledge/wiki/open-assumptions.md`;
4. `knowledge/wiki/parity/known-differences.md`.

Если источник используется для инженерного утверждения, указывать его ID. Если источник противоречит коду, не исправлять код вслепую: сначала занести расхождение в `known-differences.md`.
