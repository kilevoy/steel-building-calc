# Карточки ввода

Для будущего раздела единого здания ввод должен быть понятнее, чем набор сокращений. Инженерные обозначения не скрываются, но объясняются.

Логические блоки: район строительства, геометрия здания, нагрузки, колонны, ферма, прогоны, балка покрытия, оконные ригели, подкрановая балка и сводка.

Цель: пользователь понимает, что вводит, и видит связь между исходными данными и расчётными модулями.
