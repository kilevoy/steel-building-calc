# Обзор проекта

`steel-building-calc` — web-калькулятор металлического здания на Vite, React, TypeScript, Vitest и ESLint.

Назначение проекта — расчёт отдельных конструктивных блоков здания с ориентацией на инженерную достоверность и сопоставление с Excel/VELICAN-оракулом.

Текущие вкладки:

- Column;
- Truss;
- Purlins;
- Beam Cell;
- Window Riegel;
- Crane Beam;
- Summary.

Проект находится в стадии укрепления расчётной базы, тестов, документации и будущей подготовки раздела единого здания.
